if / elif / else — 주민번호 성별 판별 if
if는 조건이 참일 때만 코드를 실행합니다. elif(else if)는 앞 조건이 거짓일 때 추가 조건을 검사하고, else는 모든 조건이 거짓일 때 실행됩니다.
in '13'은 문자가 '13' 문자열에 포함되어 있는지를 확인합니다. 
num = '260403-3000000'
gender = num[-7]      # 뒷자리 첫 번째 문자 = 성별 코드

if gender in '13':       # '1' 또는 '3'이면
    print('남자')
elif gender in '24':    # '2' 또는 '4'이면
    print('여자')
output
남자
삼항 연산자 — 조건부 표현식 Tip
Python의 조건부 표현식(삼항 연산자)은 참값 if 조건 else 거짓값 형태로 한 줄에 조건 분기를 작성합니다.
간단한 if/else를 한 줄로 줄일 수 있어 코드가 간결해집니다. 
gender = 1

# 일반 if/else
if gender % 2:
    result = '남자'
else:
    result = '여자'

# 삼항 연산자로 한 줄에
result = '남자' if gender % 2 else '여자'
print(result)
output
남자
for + range() — 홀수 합 구하기 for
range(시작, 끝, 간격)은 숫자 범위를 생성합니다. 끝 값은 포함하지 않습니다.
range(1, 101, 2)는 1, 3, 5, …, 99 — 1부터 100 미만의 홀수를 바로 만들어 줍니다. 
# 방법 1: range + if로 홀수 필터
tot = 0
for x in range(1, 101):
    if x % 2:          # 홀수이면 True
        tot += x
print(tot)

# 방법 2: range 간격을 2로 설정 (더 간결)
tot = sum(range(1, 101, 2))
print(tot)
output
2500
2500
for + 리스트 — 합격·불합격 판별 for
for 변수 in 리스트는 리스트의 항목을 하나씩 꺼내서 사용합니다.
리스트를 읽기만 할 때는 이 방식이 가장 깔끔합니다. f-string을 사용하면 변수를 문자열 안에 쉽게 삽입할 수 있습니다. 
score = [50, 58, 67, 72, 85, 90, 99, 100]

for s in score:
    print(s, '-', '합격' if s >= 60 else '불합격')

for + else + break — 암호 입력 (3회 제한) for
break는 반복문을 즉시 탈출합니다.
for의 else는 반복이 break 없이 끝났을 때만 실행됩니다. "암호를 3번 모두 틀린 경우"처럼 정상 종료 시 후처리에 유용합니다. 
PW = 'bluesky'

for i in range(3):
    if input('암호를 입력하세요: ') == PW:
        print('환영합니다')
        break              # 맞으면 즉시 탈출
    print('다시 입력하세요')
else:
    print('관리자에게 연락하세요')  # break 없이 끝날 때만 실행

while + := (대입 연산자) — 조건과 저장 동시에 while
:= (walrus 연산자, Python 3.8+)는 값을 변수에 저장하면서 동시에 반환합니다.
while (a := input(...)) != PW는 입력값을 a에 저장하고, 동시에 비밀번호와 비교합니다.
while의 else도 break 없이 조건이 거짓이 된 경우 실행됩니다. 
PW = "bluesky"
count = 0

while (a := input("입력: ")) != PW:
    if count == 2:
        print("관리자에게 연락하세요")
        break
    print("다시 입력하세요")
    count += 1
else:
    print("환영합니다")   # 올바른 암호를 입력했을 때
올바른 암호 입력 시
환영합니다
