파이썬 자료형 분류
숫자형 (Numeric)
• int — 정수 (무한 자릿수)
• float — 실수 (8바이트 double)
• bool — True/False (int 하위 타입)
순차형 (Sequence)
• tuple — 불변 순서열
• list — 가변 순서열
• str — 문자열
• range — 정수 범위 (메모리 효율적)
iterable / iterator
• iterable: iter()로 iterator를 생성할 수 있는 객체
→ list, tuple, str, range 등
• iterator: next()로 다음 요소를 반환하는 객체
→ StopIteration이 발생할 때까지
예제 1: 정수(int) — 파이썬은 무한 자릿수! int · 무한정수
대부분의 언어(C, Java 등)는 정수의 크기가 고정되어 있지만, 파이썬의 int는 자릿수 제한이 없습니다.
내부적으로 필요한 만큼 메모리를 동적으로 할당하기 때문에, 아무리 큰 수도 정확하게 표현합니다.
주의: 숫자 뒤의 ;은 파이썬에서 선택적이지만 일반적으로는 쓰지 않습니다. 
# 파이썬의 정수는 자릿수 제한이 없는 무한 정수입니다.
# C/Java에서는 오버플로우가 나지만, 파이썬에서는 정확히 계산됩니다.
a = 100000000000000000000 + 1
print(a)         # 100000000000000000001
print(type(a))   # <class 'int'>

# 정수가 얼마나 큰 값도 처리 가능한지 확인
big = 2 ** 100    # 2의 100제곱
print(big)        # 1267650600228229401496703205376
▶ 출력
100000000000000000001
<class 'int'>
1267650600228229401496703205376
예제 2: 실수(float) — 8바이트 double 정밀도 float · 부동소수점
파이썬의 float는 C언어의 double형(8바이트, 64비트)을 사용합니다.
소수점 이하 약 15~17자리까지 표현할 수 있으며, 그 이상의 자릿수는 반올림됩니다.
따라서 3.123456789123456789처럼 입력해도 내부적으로 15~16자리로 잘립니다. 
# 파이썬 실수는 8바이트(64비트) double 정밀도
# 소수점 이하 약 15~16자리까지만 정확히 표현 가능
a = 3.123456789123456789
print(a)         # 3.123456789123457 (마지막 자리 반올림됨)
print(type(a))   # <class 'float'>

# 부동소수점 표현의 한계 — 0.1 + 0.2는 0.3이 아닐 수 있음
print(0.1 + 0.2)  # 0.30000000000000004 (IEEE 754 부동소수점 오차)
▶ 출력
3.123456789123457
<class 'float'>
0.30000000000000004
예제 3: bool — 숫자형(Numeric)의 일부! bool · True/False · 산술연산
파이썬의 bool은 int의 하위 타입입니다. True는 정수 1, False는 정수 0과 동일합니다.
이 때문에 bool 값으로 산술 연산이 가능합니다. 이는 "조건을 만족한 개수" 세기 등에 활용됩니다.
isinstance(True, int)는 True를 반환합니다 — bool이 int를 상속하기 때문입니다. 
# bool은 int의 하위 타입: True=1, False=0
a = True
print(a, type(a))         # True <class 'bool'>

# True=1, False=0 이므로 산술 연산이 가능합니다
print(True + True)         # 2  (1 + 1)
print(True * False)        # 0  (1 * 0)
print(True + 10)            # 11 (1 + 10)

# bool은 int의 하위 타입 — isinstance로 확인
print(isinstance(True, int))   # True

# 응용: 리스트에서 True인 요소 개수 세기
data = [3, -1, 5, -2, 7]
count_positive = sum(x > 0 for x in data)
print(count_positive)         # 3 (양수가 3개)
▶ 출력
True <class 'bool'>
2
0
11
True
3
예제 4: 순차형(Sequence) 4종류 한눈에 비교 tuple · list · str · range
순차형은 인덱스로 접근 가능하고 순서가 있는 자료형입니다.
tuple은 소괄호 (), list는 대괄호 [], str은 따옴표, range는 range()로 만듭니다.
네 가지 모두 for 문으로 순회(iterate)할 수 있습니다. 
a = (1, 2, 3)      # tuple — 불변(immutable): 생성 후 변경 불가
b = [1, 2, 3]      # list  — 가변(mutable): 요소 추가/삭제/변경 가능
c = "123"           # str   — 불변 문자열
d = range(1, 4)     # range — 정수 범위 객체 (1, 2, 3)

# type() 으로 각각의 자료형 확인
for x in [a, b, c, d]:
    print(x, type(x))

# 인덱스로 첫 번째 요소 접근 (모두 0부터 시작)
print(a[0], b[0], c[0])   # 1 1 1
▶ 출력
(1, 2, 3) <class 'tuple'>
[1, 2, 3] <class 'list'>
123 <class 'str'>
range(1, 4) <class 'range'>
1 1 1
예제 5: tuple 생성 방법 — 헷갈리는 규칙! tuple · 쉼표 · 단일원소
tuple을 만드는 것은 괄호가 아니라 쉼표(,)입니다!
1,은 원소가 1개인 tuple이고, (1)은 그냥 정수 1입니다.
원소가 1개인 tuple을 만들려면 반드시 (1,)처럼 쉼표를 붙여야 합니다. 
# 쉼표가 tuple을 만드는 핵심! 괄호는 선택사항
a = 1,          # (1,) — 원소 1개짜리 tuple
b = (1, )        # (1,) — 위와 동일
c = 1, 2         # (1, 2) — 원소 2개짜리 tuple
d = (1)           # 1 — 그냥 정수! tuple이 아님

for x in [a, b, c, d]:
    print(x, type(x))

# tuple 패킹/언패킹
point = 3, 5       # 패킹: (3, 5) tuple
x, y = point       # 언패킹: x=3, y=5
print(x, y)        # 3 5
▶ 출력
(1,) <class 'tuple'>
(1,) <class 'tuple'>
(1, 2) <class 'tuple'>
1 <class 'int'>
3 5
예제 6: str — 문자열을 만드는 다양한 방법 str · 이스케이프 · 멀티라인
파이썬 문자열은 ', ", ''', """ 네 가지로 만들 수 있습니다.
이스케이프 문자: \n(줄바꿈), \'(작은따옴표), \"(큰따옴표), \\(백슬래시), \(줄 이어쓰기).
따옴표 안에 다른 종류의 따옴표를 바로 쓸 수 있어, 따옴표 포함 문자열도 이스케이프 없이 표현할 수 있습니다. 
a = '123'              # 작은따옴표
b = "123"              # 큰따옴표 — a와 동일
c = "I'm a boy!"       # 큰따옴표 안에 작은따옴표 자유롭게 사용
d = 'This is "Wonderful".'  # 작은따옴표 안에 큰따옴표 사용
e = 'Good\nMorning'    # \n 은 줄바꿈 (newline)
f = '''Good
Morning'''            # 삼중따옴표 — 여러 줄 문자열 직접 작성
g = 'Good\
Morning'             # \ 끝에 두면 줄을 이어 씀 (줄바꿈 없음)
h = 'I\'m a boy!'       # 이스케이프로 작은따옴표 포함

for x in [a, b, c, d, e, f, g, h]:
    print(x, type(x))
▶ 출력
123 <class 'str'>
123 <class 'str'>
I'm a boy! <class 'str'>
This is "Wonderful". <class 'str'>
Good
Morning <class 'str'>
Good
Morning <class 'str'>
GoodMorning <class 'str'>
I'm a boy! <class 'str'>
예제 7: range — 메모리 효율적인 정수 범위 range · start/stop/step · 메모리
range(stop), range(start, stop), range(start, stop, step) 세 가지 형태로 사용합니다.
start는 기본값 0, step은 기본값 1입니다. stop만 필수입니다.
range는 실제 숫자를 모두 메모리에 저장하지 않고, start/stop/step 세 값만 저장합니다.
따라서 range(1, 10001)과 list(range(1, 10001))의 메모리 크기가 크게 다릅니다. 
# range의 세 가지 사용법
a = range(5)          # range(0, 5)  → 0,1,2,3,4
b = range(0, 5)       # range(0, 5)  → 위와 동일
c = range(0, 5, 1)    # range(0, 5)  → 위와 동일
d = range(5, 1, -1)   # range(5,1,-1) → 5,4,3,2 (역순)

for x in [a, b, c, d]:
    print(x, type(x))

# 메모리 크기 비교: range vs list
import sys
num = 10001
r = range(1, num)     # start, stop, step 3개 값만 저장
lst = list(r)          # 10000개 정수를 모두 메모리에 저장
print(sum(r), sum(lst))               # 합은 같음: 50005000
print(sys.getsizeof(r),
      sys.getsizeof(lst))           # 48 vs 80056 — 차이가 엄청남!
▶ 출력
range(0, 5) <class 'range'>
range(0, 5) <class 'range'>
range(0, 5) <class 'range'>
range(5, 1, -1) <class 'range'>
50005000 50005000
48 80056 ← range는 48바이트, list는 80056바이트!
예제 8: iterable vs iterator — for 문의 작동 원리 iterable · iterator · iter() · next()
iterable: iter()를 호출하면 iterator를 생성하는 객체. (list, tuple, str, range 등)
iterator: next()를 호출하면 다음 요소를 반환하고, 더 이상 없으면 StopIteration을 발생시킵니다.
for 문은 내부적으로 iter()로 iterator를 만들고, next()로 순서대로 꺼냅니다.
중요: list는 iterator가 아닙니다. list에 직접 next()를 쓰면 TypeError가 납니다. 
# list는 iterable이지만 iterator는 아닙니다
a = [1, 2, 3]
# next(a)  → TypeError! list는 next()를 쓸 수 없음

# iter()로 iterator 생성 후 next() 사용 가능
b = iter(a)          # list_iterator 객체 생성
print(next(b))       # 1
print(next(b))       # 2
print(next(b))       # 3
# print(next(b))   → StopIteration! 더 이상 요소 없음

# for 문은 위의 동작을 자동으로 처리합니다
# for x in a: → iter(a)로 iterator 생성 → next()로 하나씩 꺼냄 → StopIteration 시 종료
for x in a:
    print(x, end=' ')   # 1 2 3

# iterable은 iter()를 여러 번 호출할 수 있고, 호출할 때마다 새 iterator 생성
c = iter(a)
d = iter(a)
print(id(c) == id(d))  # False — 서로 다른 객체
▶ 출력
1
2
3
1 2 3
False
예제 9: input() — 사용자 입력 받기 input · map · split · iterator
input()은 항상 문자열(str)을 반환합니다. 숫자로 쓰려면 int() 등으로 변환이 필요합니다.
여러 값을 한 줄에 받을 때는 .split()으로 나누고, map(int, ...)으로 일괄 변환합니다.
map 객체는 iterator입니다. for 문에서 소진한 뒤에는 next()를 쓸 수 없습니다. 
# 1. 기본 입력 — 항상 str 반환
# a = input('숫자를 입력하세요: ')
# print(a, type(a))  → '10' <class 'str'>

# 2. 정수로 변환
# a = int(input('숫자를 입력하세요: '))
# print(a, type(a))  → 10 <class 'int'>

# 3. 여러 값 입력 → split()으로 리스트로 변환
# a = input('숫자를 여러개 입력: ').split()
# → '1 2 3'.split() = ['1', '2', '3']  (str 리스트)

# 4. map()으로 리스트 전체를 int로 일괄 변환
# a = input('숫자를 여러개 입력: ').split()
# b = map(int, a)  → map 객체 (iterator)

# map은 iterator이므로 for 후 StopIteration 됨
a = ['1', '2', '3']   # split() 결과 예시
b = map(int, a)       # map 객체 — iterator
print(type(b))         # <class 'map'>
for x in b:            # for로 소진
    print(x, type(x), end=' ')
# next(b) → StopIteration! for에서 이미 소진됨
▶ 출력
<class 'map'>
1 <class 'int'> 2 <class 'int'> 3 <class 'int'>
