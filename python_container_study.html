예제 1: list vs tuple — 가변 vs 불변, 메모리 크기 list · tuple · getsizeof · dir
list는 가변(mutable), tuple은 불변(immutable)입니다.
불변인 tuple이 list보다 메모리를 적게 사용합니다.
dir()로 확인하면 tuple에는 count, index 두 개뿐이지만,
list에는 append, sort, pop 등 수정 관련 메서드가 추가로 있습니다. 
a = (1, 2, 3)   # tuple
b = [1, 2, 3]   # list
import sys
print(sys.getsizeof(a), sys.getsizeof(b))   # 64  88

# __로 시작하지 않는 메서드만 확인
dira = [x for x in dir(a) if not x.startswith('__')]
dirb = [x for x in dir(b) if not x.startswith('__')]
print('tuple', dira)
print('list',  dirb)
▶ 출력
64 88
tuple ['count', 'index']
list ['append', 'clear', 'copy', 'count', 'extend', 'index', 'insert', 'pop', 'remove', 'reverse', 'sort']
예제 2: index — 양수 인덱스 & 음수 인덱스 index · subscription · IndexError
양수 인덱스: 0부터 시작, 왼쪽 → 오른쪽.
음수 인덱스: -1부터 시작(마지막 item), 오른쪽 → 왼쪽.
subscription: 하나의 index만 사용하는 연산 — a[1]
범위를 벗어난 index 사용 시 IndexError 발생. 
a = ['Morning', 'Afternoon', 'Evening']
#    인덱스:   0            1             2
#    음수:    -3           -2            -1
print(a[0], a[-1])   # Morning  Evening

num = '260403-3000000'
#      0123456 7890123
print(num[7], num[-7])   # 3  3  (동일한 위치)

# print(num[20])  → IndexError: string index out of range
▶ 출력
Morning Evening
3 3
예제 3: slicing — start:stop:step slice · __getitem__ · __setitem__
슬라이싱: a[start:stop:step] 형식. 범위를 벗어나도 오류 없이 빈 값 반환.
기본값: start=0, stop=len(객체), step=1
내부적으로 __getitem__(읽기)과 __setitem__(쓰기)을 호출합니다. 
a = [10, 20, 30, 40, 50]

# __getitem__ (읽기)
print(a[:])      # [10,20,30,40,50]  전체 복사
print(a[:3])    # [10,20,30]        앞에서 3개
print(a[-2:])   # [40,50]           뒤에서 2개
print(a[::-1])  # [50,40,30,20,10]  역순
print(a[:-1])   # [10,20,30,40]     마지막 제외

# __setitem__ (쓰기, list만 가능 - 가변객체)
a[3] = 'new'
print(a)        # [10, 20, 30, 'new', 50]

a[2:3] = ['x', 'y', 'z']    # 슬라이스에 여러 값 할당
print(a)        # [10, 20, 'x', 'y', 'z', 'new', 50]

a[0] = ['a', 'b']           # item으로 리스트 자체를 넣을 수도 있음
print(a)        # [['a','b'], 20, 'x', 'y', 'z', 'new', 50]

# 실용 예시: 주민번호에서 날짜 추출
num = '260403-3000000'
year  = '20' + num[0:2]   # '2026'
month = num[2:4]            # '04'
day   = num[4:6]            # '03'
print(year, month, day)    # 2026 04 03
▶ 출력
[10, 20, 30, 40, 50]  [10, 20, 30]  [40, 50]  [50, 40, 30, 20, 10]  [10, 20, 30, 40]
[10, 20, 30, 'new', 50]
[10, 20, 'x', 'y', 'z', 'new', 50]
[['a', 'b'], 20, 'x', 'y', 'z', 'new', 50]
2026 04 03
예제 4: + 연결 연산 · * 복사 연산 + 연결 · * 복사 · 다형성
+ (연결): 같은 타입 두 객체를 연결해 동일 타입의 새 객체 생성.
* (복사): 객체 * 정수 형식으로 item을 정수만큼 반복 복사.
같은 연산자라도 타입에 따라 다르게 동작하는 것이 다형성(Polymorphism)입니다. 
# + 연결 연산
def add(a, b): return a + b

print(add(10, 20))          # 30         int 덧셈
print(add('Good', 'Morning'))  # GoodMorning  str 연결
print(add([1,2], [3,4]))   # [1, 2, 3, 4] list 연결
print((1,2,3) + (4,5,6))   # (1,2,3,4,5,6) tuple 연결

# * 복사 연산
def mul(a, b): return a * b

print(mul(10, 3))           # 30          int 곱셈
print(mul('*', 3))          # ***         str 반복
print(mul((3,), 3))         # (3, 3, 3)   tuple 반복
print(mul([0], 4))          # [0,0,0,0]   list 반복
▶ 출력
30  GoodMorning  [1, 2, 3, 4]  (1, 2, 3, 4, 5, 6)
30  ***  (3, 3, 3)  [0, 0, 0, 0]
예제 5: str 주요 메서드 — split / join / replace / strip str · split · join · replace · strip
str은 불변 객체로, 메서드는 항상 새 문자열을 반환합니다(원본 변경 없음).
split(): 공백/구분자로 쪼개기 → list 반환
join(): 리스트 item을 구분자로 연결 → str 반환
strip(): 양쪽 공백(또는 지정 문자) 제거 
a = 'Good     Morning!'
print(a.endswith('ing!'), a.startswith('Good'))  # True True
print(a.split())           # ['Good','Morning!']  연속 공백 자동 처리
print(a.split(' '))        # ['Good','','','','',' Morning!']  ← 나쁜 방법!

date = '2026-04-03'
print(date.split('-'))          # ['2026','04','03']  왼→오른쪽 전부
print(date.rsplit('-', maxsplit=1))  # ['2026-04','03']  오른쪽에서 1번

# replace vs join+split (동일 결과)
print(date.replace('-', '/'))         # 2026/04/03
print('/'.join(date.split('-')))      # 2026/04/03

# strip: 양쪽 공백 제거
s = '      a    b    c     '
print(s.split())    # ['a','b','c']
print(s.strip())    # 'a    b    c'  (양쪽 공백만 제거)

# strip에 문자 지정: 해당 문자 제거
url = 'www.rapa.com'
print(url.split('.')[1])    # 'rapa'
print(url.strip('wcom.'))   # 'rapa'

# 공백 완전 제거
name = 'Yoon So  Young'
print(name.replace(' ', ''))   # YoonSoYoung
print(''.join(name.split()))   # YoonSoYoung
▶ 출력
True True
['Good', 'Morning!']
['Good', '', '', '', '', 'Morning!']
['2026', '04', '03']   ['2026-04', '03']
2026/04/03   2026/04/03
['a', 'b', 'c']   'a b c'
rapa   rapa
YoonSoYoung   YoonSoYoung
예제 6: set — 중복 제거 · hashable · hash 동치 set · hashable · hash · 중복
set은 중복을 허용하지 않습니다. item은 반드시 hashable(불변 객체)이어야 합니다.
1 == True == 1.0이고 hash 값도 같으므로, set은 이 셋을 동일 item으로 취급합니다.
빈 set는 반드시 set()으로 생성해야 합니다({}는 dict). 
# 중복 자동 제거
b = {1, 1, 1, 2, 2, 2}
print(b)    # {1, 2}

# 1, True, 1.0은 hash가 같아 → 동일 item 취급
c = {1, True, 1.0}
print(len(c), c)   # 1  {1}

if 1 == True:  print(hash(1), hash(True))   # 1 1
if 1 == 1.0:   print(hash(1), hash(1.0))    # 1 1

# 빈 set 생성: {} 아닌 set() 사용!
empty_dict = {}         # dict
empty_set  = set()      # set
print(type(empty_dict), type(empty_set))

# hashable 불가 → list는 set의 item이 될 수 없음
# {[1,2], [3,4]}  → TypeError: unhashable type: 'list'

# set 활용: 집합 연산
s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}
print(s1 | s2)   # {1,2,3,4,5,6}  합집합
print(s1 & s2)   # {3,4}          교집합
print(s1 - s2)   # {1,2}          차집합
▶ 출력
{1, 2}
1 {1}
1 1    1 1
<class 'dict'> <class 'set'>
{1, 2, 3, 4, 5, 6}   {3, 4}   {1, 2}
예제 7: dict — 생성 · 추가 · 갱신 · 삭제 · 조회 dict · key · value · get · keys · items
dict는 key: value 쌍으로 데이터를 저장합니다.
key는 hashable(불변 객체)이어야 하며 중복 불가. value는 제약 없음.
a[key]로 없는 key 접근 시 KeyError, a.get(key)는 None을 반환해 안전합니다. 
# 두 가지 생성 방법 (동일 결과)
a = {'A': 65, 'B': 66}
b = dict(A=65, B=66)
print(a, b)   # {'A':65,'B':66}  {'A':65,'B':66}

a = {'age': 20}
a['name'] = 'YoonSoYoung'  # 추가
print(a)   # {'age':20, 'name':'YoonSoYoung'}
a['age']  = 30               # 갱신
print(a)   # {'age':30, 'name':'YoonSoYoung'}
del a['age']                  # 삭제
print(a)   # {'name':'YoonSoYoung'}

a = {'age': 30, 'name': 'YoonSoYoung'}
print(a['age'])          # 30
# a['height']            → KeyError!
print(a.get('height'))  # None  (안전한 조회)
print(a.keys())          # dict_keys(['age','name'])
print(a.values())        # dict_values([30,'YoonSoYoung'])
print(a.items())         # dict_items([('age',30),('name','YoonSoYoung')])
for x in a:
    print(x, end=' ')    # age name  (key만 순회)
▶ 출력
{'A': 65, 'B': 66} {'A': 65, 'B': 66}
{'age': 20, 'name': 'YoonSoYoung'}
{'age': 30, 'name': 'YoonSoYoung'}
{'name': 'YoonSoYoung'}
30   None
dict_keys(['age', 'name'])
dict_values([30, 'YoonSoYoung'])
dict_items([('age', 30), ('name', 'YoonSoYoung')])
age name
