예제 1: map() — 함수를 각 요소에 적용 map · iterator · next
map(함수, 이터러블)은 이터러블의 각 요소에 함수를 적용한 결과를 lazy하게 반환하는 map 객체를 만듭니다.
list()나 next()로 값을 꺼내야 실제 계산이 됩니다.
여러 이터러블을 넘기면 같은 위치의 요소끼리 함수에 전달됩니다. 
def func(a, b):
    return a + b

x = range(1, 10, 2)  # 1, 3, 5, 7, 9
y = range(2, 11, 2)  # 2, 4, 6, 8, 10
z = map(func, x, y)  # map 객체 생성 (계산은 아직 X)
print(list(z))        # list()로 변환해야 실제 계산됨

# 문자열 길이 구하기
countries = ['Korea', 'USA', 'Philippines', 'China']
lengths = list(map(len, countries))
print(lengths)
▶ 출력
[3, 7, 11, 15, 19]
[5, 3, 11, 5]
예제 2: map() + * 언패킹 — 2D 리스트 활용 map · min · unpack · *
*a는 리스트를 언패킹하여 여러 인자로 전달합니다.
map(min, *a)는 2D 리스트 a의 각 열(column)에서 최솟값을 구합니다.
zip(*a)와 함께 쓰면 행과 열을 전치(transpose)할 수 있습니다. 
a = [[3, 4, 1, 2],
     [7, 2, 6, 3],
     [5, 1, 8, 2]]

# 각 열에서 최솟값 구하기 (열 기준)
print(list(map(min, *a)))   # [3,1,1,2]

# zip(*a)로 행열 전치 (transpose)
print(list(map(list, zip(*a))))
▶ 출력
[3, 1, 1, 2]
[[3, 7, 5], [4, 2, 1], [1, 6, 8], [2, 3, 2]]
예제 3: filter() — 조건에 맞는 요소만 선택 filter · bool · 조건함수
filter(함수, 이터러블)은 함수가 True를 반환하는 요소만 통과시킵니다.
함수 자리에 None을 넣으면 참으로 평가되는 값만 남깁니다.
map과 마찬가지로 지연 평가(lazy) 방식입니다. 
a = [10, 3, 6, 7, 9, 30, 11, 20]

def is_even(x):
    return x % 2 == 0  # 짝수이면 True

result = filter(is_even, a)
print(list(result))   # 짝수만 남김
▶ 출력
[10, 6, 30, 20]
예제 4: zip() — 여러 이터러블을 묶기 zip · tuple · 병렬순회
zip(iter1, iter2, ...)은 각 이터러블의 같은 위치 요소를 tuple로 묶어 반환합니다.
가장 짧은 이터러블을 기준으로 멈춥니다.
zip(*a)를 사용하면 2D 리스트를 전치(transpose)할 수 있습니다. 
# 기본 zip
print(list(zip('abcdefg', range(3), range(4))))
# 가장 짧은 range(3) 기준으로 3개만

# dict comprehension과 조합
keys   = ['a', 'b', 'c', 'd', 'e']
values = [1, 2, 3, 4, 5]
d = {k:v for k, v in zip(keys, values)}
print(d)
▶ 출력
[('a', 0, 0), ('b', 1, 1), ('c', 2, 2)]
{'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5}
예제 5: Generator Expression — 지연 평가 이터레이터 generator · next · StopIteration
( )를 사용한 컴프리헨션은 generator 객체를 반환합니다.
generator는 next()를 호출할 때마다 값을 하나씩 계산합니다.
모든 값을 소진하면 StopIteration 예외가 발생합니다.
메모리를 한꺼번에 사용하지 않아 대용량 데이터에 유리합니다. 
a = [1, 3, 5, 7]
b = (x for x in a)  # generator 객체 생성
print(b)             # <generator object>

print(next(b))  # 1
print(next(b))  # 3
print(next(b))  # 5
print(next(b))  # 7
# print(next(b))  → StopIteration: 더 이상 값 없음
▶ 출력
<generator object <genexpr> at 0x...>
1
3
5
7
예제 6: List / Set Comprehension 기본 list comp · set comp · [ ] · { }
[ ] → list, { } → set을 반환합니다.
문법: [식 for 변수 in 이터러블]
set은 중복을 자동으로 제거하고 순서가 없습니다. 
a = [1, 3, 5, 7]

b = [x for x in a]  # list comprehension
c = {x for x in a}  # set  comprehension

print(b, type(b))  # [1,3,5,7] <class 'list'>
print(c, type(c))  # {1,3,5,7} <class 'set'>

# 제곱값 리스트
squares = [2**i for i in range(1, 10)]
print(squares)
▶ 출력
[1, 3, 5, 7] <class 'list'>
{1, 3, 5, 7} <class 'set'>
[2, 4, 8, 16, 32, 64, 128, 256, 512]
예제 7: Comprehension + if 조건 — 필터링 list comp · if · filter
[식 for 변수 in 이터러블 if 조건]
조건이 True인 요소만 포함됩니다. filter()와 동일한 효과입니다.
for 뒤에 if를 붙이면 필터링, 식(표현식) 안에 if를 넣으면 삼항 조건식입니다. 
a = [3, 7, 8, 1, 2, 4, 9, 10, 5, 6]

# 짝수만 제곱 — for 문 버전
b = []
for x in a:
    if x % 2 == 0:
        b.append(x**2)

# 짝수만 제곱 — comprehension 버전 (한 줄!)
b = [x**2 for x in a if x % 2 == 0]
print(b)

# 길이 3 이상인 단어만 대문자로
w = ['cat', 'hi', 'a', 'dog', 'tiger', 'no']
result = [x.upper() for x in w if len(x) >= 3]
print(result)
▶ 출력
[64, 4, 16, 100, 36]
['CAT', 'DOG', 'TIGER']
예제 8: Dict Comprehension — 딕셔너리 생성/변환 dict comp · items · k:v
{ key: value for ... } 형식으로 딕셔너리를 만듭니다.
.items()를 활용하면 기존 딕셔너리의 키-값을 뒤집거나 변환할 수 있습니다.
문법: {k:v for k, v in 딕셔너리.items()} 
# zip으로 딕셔너리 생성
d = {k:v for k, v in zip("abcde", range(1, 6))}
print(d)

# 딕셔너리 키-값 뒤집기
a = dict(A=1, B=2, C=3, D=4, E=5)
b = {v:k for k, v in a.items()}  # 키↔값 교환
print(b)
▶ 출력
{'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5}
{1: 'A', 2: 'B', 3: 'C', 4: 'D', 5: 'E'}
예제 9: str.join() + Generator — 문자열 필터링 join · generator · 문자열
''.join(이터러블)은 이터러블의 문자열 요소를 하나로 합칩니다.
generator expression을 직접 인자로 넣으면 중간 리스트 없이 메모리를 효율적으로 사용합니다. 
a = 'dataanalysis'
# 모음(aeiou)만 추출하여 문자열로 합치기
b = ''.join(x for x in a if x in "aeiou")
print(b)  # 'aaaai'

# 월 이름 앞 3글자 리스트
months = ['January', 'February', 'March', 'April',
         'May', 'June', 'July', 'August',
         'September', 'October', 'November', 'December']
month = [x[:3] for x in months]
print(month)
▶ 출력
aaaai
['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec']

📊 map/filter vs Comprehension 비교

방식문법 예시결과 타입특징
map()list(map(fn, a))map 객체 → listlazy, 외부 함수 필요
filter()list(filter(fn, a))filter 객체 → listlazy, True 요소만
List Comp[fn(x) for x in a]list (즉시)가독성 최고, 단순
Set Comp{fn(x) for x in a}set (즉시)중복 제거
Dict Comp{k:v for k,v in ...}dict (즉시)키-값 쌍 생성
Generator Exp(fn(x) for x in a)generator (lazy)메모리 효율
