예제 1: positional argument vs keyword argument argument · positional · keyword
positional argument: func(10, 20, 30) — 순서(위치)대로 parameter에 전달됩니다.
keyword argument: func(a=10, b=20) — parameter 이름을 지정하므로 순서에 상관없습니다.
p와 k를 혼합할 때는 positional이 앞, keyword가 뒤에 와야 합니다. 
def func(a, b, c):
    print([a, b, c])

func(10, 20, 30)          # positional: 순서대로
func(a=30, b=10, c=20)   # keyword: 순서 무관
func(c=30, a=10, b=20)   # keyword: 다른 순서도 OK
func(10, c=30, b=20)      # 혼합: p 먼저, k 나중
output
[10, 20, 30]
[30, 10, 20]
[10, 20, 30]
[10, 20, 30]
예제 2: default argument — 기본값 설정 default · non-default
default argument: parameter에 기본값을 지정 — 호출 시 생략 가능합니다.
non-default argument: 기본값 없음 — 반드시 argument를 전달해야 합니다.
정의 시 규칙: non-default 먼저, default 나중에 위치해야 합니다. 
def func(a=10, b=20, c=30):
    print([a, b, c])

func()                    # 모두 기본값 사용
func(b=40)               # b만 변경
func(c=100, b=10)       # b, c 변경 (순서 무관)
func(100)                 # a만 변경
func(100, 200)            # a, b 변경
func(a=100, c=200)      # a, c 변경
output
[10, 20, 30]
[10, 40, 30]
[10, 10, 100]
[100, 20, 30]
[100, 200, 30]
[100, 20, 200]
예제 3: *args — 가변 positional parameter var positional · *args · tuple
*parameter: 0개 이상의 positional argument를 하나의 tuple로 묶어 받습니다.
몇 개의 인자를 전달할지 미리 알 수 없을 때 사용합니다.
*a 뒤에 오는 parameter는 자동으로 keyword only가 됩니다. 
def func(*a):
    print(a, type(a))

func()            # 인자 없음
func(1)           # 1개
func(1, 2, 3)    # 3개 → tuple로 묶임

# *a 뒤의 b, c는 keyword only가 됨
def func2(*a, b, c):
    print([a, b, c])

func2(10, 20, 30, b=100, c=200)
output
() <class 'tuple'>
(1,) <class 'tuple'>
(1, 2, 3) <class 'tuple'>
[(10, 20, 30), 100, 200]
예제 4: **kwargs — 가변 keyword parameter var keyword · **kwargs · dict
**parameter: 0개 이상의 keyword argument를 하나의 dict로 묶어 받습니다.
key는 식별자 규칙을 만족해야 합니다 (숫자로 시작 불가).
**kwargs는 항상 parameter 목록의 맨 마지막에 위치해야 합니다. 
def func(**a):
    print(a, type(a))

func()                # 인자 없음
func(A=1)            # 1개
func(A=1, B=2)      # 2개 → dict로 묶임
output
{} <class 'dict'>
{'A': 1} <class 'dict'>
{'A': 1, 'B': 2} <class 'dict'>
예제 5: positional only ( / ) — keyword 사용 금지 positional only · /
,/ 기호 앞에 있는 parameter는 positional argument만 허용합니다.
keyword argument(a=10)로 호출하면 TypeError가 발생합니다.
파이썬 내장함수 sorted()의 iterable이 이 방식입니다. 
def func(a, /):
    print(a)

func(10)      # OK: positional
# func(a=10)  → TypeError: positional-only 위반

# 혼합: a는 positional only, b는 p or k, c는 keyword only
def func2(a, /, b, *, c):
    print([a, b, c])

func2(1, 2, c=3)
func2(1, b=2, c=3)   # b는 keyword도 OK
output
10
[1, 2, 3]
[1, 2, 3]
예제 6: keyword only ( * ) — positional 사용 금지 keyword only · *
*, 기호 뒤에 있는 parameter는 keyword argument만 허용합니다.
positional argument(func(10))로 전달하면 TypeError가 발생합니다.
sorted()의 key와 reverse가 keyword only의 대표 사례입니다. 
def func(*, a):
    print(a)

func(a=10)   # OK: keyword
# func(10)  → TypeError: 0 positional arguments (*, 이후는 k only)

# 실제 예시: sorted() 내장 함수
# sorted(iterable, /, *, key=None, reverse=False)
nums = [3, 1, 4, 1, 5]
print(sorted(nums, reverse=True))   # reverse는 keyword only
output
10
[5, 4, 3, 1, 1]
예제 7: return — 값 반환과 None return · None · bool
return을 생략하거나 return만 쓰면 None이 반환됩니다.
None은 print()로 출력 가능하며, 조건 검사에서 False로 취급됩니다.
반환값을 변수에 저장하거나 다른 식에서 바로 활용할 수 있습니다. 
def add(a, b):
    return a + b       # 값 반환

def show(a):
    print(a)            # return 없음 → None 반환

print(add(3, 5))      # 8
x = show(10)          # None 반환
print(x)              # None

# None은 False 취급
print("참" if show(10) else "거짓")
output
8
10
None
10
거짓
예제 8: lambda — 이름 없는 함수 (expression) lambda · anonymous function
lambda는 def 없이 한 줄로 함수를 표현합니다.
문법: lambda 매개변수 : 표현식 — return 없이 표현식이 자동으로 반환됩니다.
변수에 저장하거나 다른 함수(map, filter, sorted)에 직접 인자로 넘길 수 있습니다. 
# 제곱 함수
square = lambda x : x**2
print(square(5))      # 25

# 삼항 조건식 활용
parity = lambda x : '짝수' if x % 2 == 0 else '홀수'
print(parity(10))     # 짝수
print(parity(5))      # 홀수

# 두 수의 합
add = lambda x, y : x + y
print(add(10, 20))    # 30
output
25
짝수
홀수
30
예제 9: lambda + map / sorted — 실전 활용 lambda · map · sorted · key
map(lambda ..., iterable): 각 요소에 lambda를 적용합니다.
sorted(iterable, key=lambda ...): 정렬 기준을 lambda로 지정합니다.
lambda는 별도 함수를 정의하지 않고 한 번만 쓰는 간단한 함수에 가장 적합합니다. 
a = [1, 2, 3, 4, 5]
b = [2, 4, 6, 8, 10]

# map + lambda: 같은 위치 요소끼리 더하기
c = list(map(lambda x, y: x + y, a, b))
print(c)

# sorted + key: 절댓값 기준 정렬
nums = [-3, 1, -7, 2, -1]
print(sorted(nums, key=lambda x: abs(x)))

# sorted + key: 문자열 길이 기준 정렬
words = ['banana', 'apple', 'fig', 'cherry']
print(sorted(words, key=lambda x: len(x)))
output
[3, 6, 9, 12, 15]
[1, -1, 2, -3, -7]
['fig', 'apple', 'banana', 'cherry']

순서종류기호예시

②positional or keyword(기호 없음)def f(b)
③var positional*parameterdef f(*c)
④keyword only*, parameterdef f(*, d)
⑤var keyword**parameterdef f(**e)
